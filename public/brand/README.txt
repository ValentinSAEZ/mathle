Placez ici vos fichiers de logo/brand pour qu’ils s’affichent dans toute l’application.

Recommandé (tous facultatifs, mais plus vous en mettez, mieux c’est) :
- logo.png            -> utilisé dans l’en‑tête, titres, pages offline/auth
- logo.svg            -> (si disponible) utilisé comme source préférée dans Auth
- logo-192.png        -> icône PWA 192×192 (manifest.json)
- logo-512.png        -> icône PWA 512×512 (manifest.json)
- favicon-32.png      -> favicon 32×32 (index.html)
- favicon-16.png      -> favicon 16×16 (index.html)
- apple-touch-icon.png -> 180×180 pour iOS (index.html)

Après avoir copié vos fichiers, faites un rafraîchissement dur du navigateur (Cmd+Shift+R) pour contourner le cache.
